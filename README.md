<h1 align="center"> 💻 Desafio amigo secreto Alura 💻</h1>

Desenvolvimento de uma aplicação para sorteio aleatório de nomes inseridos em uma lista pelo usuário

<h2 align="center"> 💡 Tecnologia aplicada </h2>
Javascript, CSS e HTML.


<h3 align="center">🔨 Funcionalidades </h3>

1. Adicionar nomes à lista;

2. Atualizar a lista de amigos;

3. Verificar se há nomes na lista;

4. Sortear um amigo aleatóriamente.
